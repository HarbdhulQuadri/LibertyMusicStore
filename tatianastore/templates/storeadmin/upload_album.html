Upload an album

  • The album must be ZIP file. The user can download this exact ZIP file after paying for the content.
  • The album must contain songs as MP3 files. Make sure the filenames are in the order they appear on the album. E.g. 01-first-song.mp3, 02-second-song.mp3
  • The album can bought as a whole (user downloads ZIP file) or each MP3 separately
  • The MP3 files must have ID3 tag meta information which contains the song title
  • The ZIP file must contain at least one JPEG image which will be used as the album cover art

You can edit all files, information, prices and song order after the upload.

Warning: The upload and processing song files may take up to 10 minutes. Just don't close the page when the files are being processed.
